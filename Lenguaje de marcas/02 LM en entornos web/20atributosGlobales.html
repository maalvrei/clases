ATRIBUTOS GLOBALES

tabindex / accesskey

Los enlaces y los campos de formularios son accesibles con la tecla TAB por defecto. Con accesskey puedes ir al enlace con combinación de teclas

Con el teclado (Mayúsc+ALT+P / ALT+P dependiendo del navegador) puede poner el foco en este elemento gracias a accesskey. Al usar la tecla TAB para moverte entre enlaces y campos incluimos a otros tipos de etiquetas y le ponemos un orden con tabindex

class / id

Permite asignar una serie de propiedades CSS comunes con otras etiquetas (class) o únicas para este elemento (id)
